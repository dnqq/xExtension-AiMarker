# 安装说明

## 先决条件

- 一个安装好的 FreshRSS 实例
- 有效的 OpenAI API 密钥
- PHP 支持 curl 扩展

## 安装步骤

1. 下载此扩展的ZIP压缩包
2. 解压到 FreshRSS 的 `./extensions` 目录
3. 在 FreshRSS 管理面板的扩展管理页面中启用 "AI Marker" 扩展
4. 配置扩展:
   - 输入你的 OpenAI API 密钥
   - 如需使用代理，请设置代理 URL
   - 选择合适的 AI 模型
   - 自定义系统提示词以符合你的过滤需求

## 使用方法

一旦配置完成，扩展会自动工作。当新文章到达时，扩展会使用 LLM 分析文章内容，并将被判断为不值得阅读的文章自动标记为已读。

## 故障排除

如果扩展未正常工作:

1. 检查 FreshRSS 日志文件查看可能的错误消息
2. 确认 API 密钥有效且具有足够的额度
3. 如果使用代理，确保代理 URL 正确且可访问
4. 尝试简化系统提示词
